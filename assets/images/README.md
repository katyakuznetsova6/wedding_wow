# Фотографии для фона

В корне этой папки лежат файлы, на которые ссылается `index.html`:

| Файл | Где используется |
|------|------------------|
| `hero-1.png` … `hero-3.png` | Первый экран: анимация и подложка |
| `scene-countdown.png` | Блок с датой и таймером |
| `scene-location.png` | Локация |
| `scene-timeline.png` | Тайминг дня |
| `scene-dress.png` | Дресс-код |
| `scene-full.png` | Полноэкранное фото |
| `scene-rsvp.png` | Подтверждение участия |
| `scene-contacts.png` | Контакты организатора |
| `scene-final.png` | Финальный блок |

При замене снимков сохраняйте те же имена или обновите пути в `index.html` (атрибуты `style="--scene-bg:..."` и `src` у `<img>` в герое).
